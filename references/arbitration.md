# 劳动仲裁流程参考

## 一、先试劳动监察投诉

比仲裁快，适合欠薪、社保问题。

- 去**公司注册地**或**实际工作地**的人力资源和社会保障局
- 拨打 **12333** 投诉
- 适合的问题：拖欠工资、未缴社保、未签劳动合同
- 劳动监察有权责令公司限期支付，公司不执行可被罚款
- 周期：通常2-4周有结果

## 二、申请仲裁

### 去哪里
- **公司注册地**的劳动人事争议仲裁委员会
- 或**劳动合同履行地**的劳动人事争议仲裁委员会
- 两个地方都可以，选离自己近的

### 需要什么材料
1. **仲裁申请书**（一式两份）
   - 申请人信息：姓名、身份证号、住址、电话
   - 被申请人信息：公司全称、地址、法定代表人、电话
   - 仲裁请求：具体要多少钱、每项怎么算出来的
   - 事实和理由
2. **身份证复印件**
3. **证据清单及复印件**（一式两份）
4. **公司工商信息**（国家企业信用信息公示系统可查）

### 仲裁申请书怎么写
不用太复杂，关键是**仲裁请求要明确**：

示例：
```
仲裁请求：
1. 被申请人支付违法解除劳动合同赔偿金 XXXXX 元
   （计算方式：月平均工资XXXX元 × 工作年限X年 × 2 = XXXXX元）
2. 被申请人支付未休年假工资 XXXX 元
3. 被申请人补缴202X年X月至202X年X月的社会保险

事实和理由：
申请人于202X年X月X日入职被申请人处，担任XX岗位。
202X年X月X日，被申请人在未提前通知、未协商的情况下
单方面口头通知解除劳动合同，且未出具书面解除通知。
申请人的行为不构成《劳动合同法》第39条规定的任何情形，
被申请人的解除行为违法，应依据第87条支付2倍经济补偿。
```

### 费用
- **仲裁不收费**
- 如果请律师，律师费自己出（但符合法律援助条件的可免费）

### 周期
- 受理后45日内结案
- 案情复杂可延长15日
- 实际通常2-4个月

## 三、仲裁流程

```
提交申请
  ↓
受理（5日内决定是否受理）
  ↓
送达申请书副本给公司（5日内）
  ↓
答辩期（10日）
  ↓
开庭前调解
  ↓
调解成功 → 调解书（有法律效力）
调解不成 → 开庭审理
  ↓
裁决（45日内，复杂+15日）
  ↓
不服裁决 → 15日内向法院起诉
服裁决 → 裁决生效，可申请强制执行
```

## 四、开庭注意事项

### 仲裁员会问什么
1. 什么时候入职的？做什么岗位？
2. 月工资多少？工资结构是什么？
3. 为什么离职？公司怎么跟你说的？
4. 有没有收到书面解除通知？
5. 有没有签过协商解除协议？
6. 你有什么证据？

### 怎么回答
- 据实说，不要夸大也不要隐瞒
- 不确定的事情说"我不确定"，不要猜
- 公司律师可能会施压，保持冷静
- 回答问题简短直接，不要主动扩展
- 仲裁员问什么答什么，不要跑题

### 常见坑
- **公司说"你是自己辞职的"** → 需要你举证是被辞退（聊天记录、录音很重要）
- **公司说"你签字同意了"** → 如果签的是协商解除且金额明显不合理，可以主张重大误解或显失公平
- **公司说"你试用期不合格"** → 公司需要举证你不合格，不是你说合格就合格

## 五、裁决后

### 如果赢了
- 公司15日内不支付 → 向法院申请强制执行
- 强制执行去**公司注册地**的基层人民法院
- 需要提交：仲裁裁决书 + 强制执行申请书

### 如果输了
- 15日内向法院起诉
- 一审 → 不服 → 15日内上诉 → 二审（终审）

### 如果公司说没钱
- 申请法院查封公司账户
- 如果公司股东未实缴出资，可以追加股东为被执行人
- 如果公司是空壳，可以申请调查公司财产线索

## 六、特别注意事项

1. **不要先签字再想** — 签了协商解除协议就很难反悔
2. **不要主动写辞职信** — 自己辞职没有补偿金，公司让你写辞职信是套路
3. **保留所有沟通记录** — 微信截图、邮件转发到个人邮箱、录音
4. **不要拖太久** — 仲裁时效1年
5. **口头承诺不算数** — 公司口头答应的条件一定要落实到书面
6. **冷静处理** — 不要在群里发过激言论，不要删公司文件，不要拒不移交工作，这些可能反过来被公司利用
