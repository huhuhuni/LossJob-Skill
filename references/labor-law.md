# 劳动法维权参考

## 一、经济补偿金（N）

### 法律依据
《劳动合同法》第46条、第47条

### 计算规则
- 每满1年支付1个月工资
- 6个月以上不满1年，按1年算
- 不满6个月，支付半个月工资
- 月工资 = 离职前12个月的平均工资（包含基本工资、绩效、奖金、津贴补贴等所有货币性收入）
- 月工资高于当地上年度职工月平均工资3倍的，按3倍封顶，年限最高12年

### 适用情形（公司合法解除需给N的）
- 第36条：协商一致解除（公司提出）
- 第40条：无过失辞退（三种情形）
  - 1. 医疗期满不能从事原工作也不能从事另行安排的工作
  - 2. 不能胜任工作，经培训或调岗后仍不能胜任
  - 3. 客观情况发生重大变化，合同无法履行，达不成变更协议
- 第41条：经济性裁员
- 第44条：合同期满（公司不续签或降低条件续签员工不同意）

## 二、赔偿金（2N）

### 法律依据
《劳动合同法》第48条、第87条

### 计算规则
- 违法解除劳动合同的，按经济补偿标准的2倍支付
- 即 2N

### 常见违法解除情形
- 没有合法理由直接辞退
- 以"不能胜任"为由辞退但未经过培训或调岗
- 以"旷工"为由辞退但实际不构成旷工
- 孕期、产期、哺乳期内辞退（女职工特殊保护）
- 工伤医疗期内辞退
- 在公司连续工作满15年且距法定退休年龄不足5年辞退
- 试用期违法辞退（试用期也要有法定理由）

## 三、代通知金（+1）

### 法律依据
《劳动合同法》第40条

### 适用条件
仅限第40条的三种情形，且公司未提前30天书面通知：
1. 医疗期满不能工作
2. 不能胜任工作
3. 客观情况重大变化

### 注意
- 只有这三种情形才有+1
- 协商解除、经济性裁员没有法定的+1（但可以谈）
- +1的金额 = 1个月工资（按上月工资标准，不是12个月平均）

## 四、常见场景赔偿速算

| 场景 | 赔偿标准 | 备注 |
|------|----------|------|
| 协商解除（公司提出） | N起，可谈 | 最低N，实际可以谈更高 |
| 公司合法裁员 | N 或 N+1 | 看是否提前30天通知 |
| 公司违法解除 | 2N | 可以要求继续履行合同或拿2N |
| 合同到期公司不续签 | N | 2008年之后的工作年限 |
| 自己辞职 | 0 | 除非公司有违法行为 |
| 试用期违法辞退 | 2N | 试用期也受法律保护 |
| 公司未签劳动合同 | 2倍工资 | 入职第2个月起算，最长11个月 |

## 五、赔偿金个人所得税

### 法律依据
财税〔2018〕164号

### 计算规则
- 经济补偿金在当地上年职工平均工资3倍以内的部分：**免征个人所得税**
- 超过3倍的部分：视为一次性收入，除以工作年限（最高12年）得到月均收入，再按月度税率表计算

### 实际情况
绝大多数人的赔偿金在3倍平均工资以内，**不需要缴税**。

以北京为例（2024年数据）：
- 北京上年职工平均工资约13万/年
- 3倍 ≈ 39万
- 赔偿金在39万以内免个税

### 超过3倍的计税方法
1. 超出部分 ÷ 工作年限 = 月均收入
2. 月均收入 - 5000 = 应纳税所得额
3. 按综合所得税率表计算月均税额
4. 月均税额 × 工作年限 = 总税额

## 六、不能被裁的人

《劳动合同法》第42条，以下情形公司不得依据第40条、第41条解除：

- 从事接触职业病危害作业的劳动者未进行离岗前职业健康检查
- 疑似职业病病人在诊断或医学观察期间
- 患病或非因工负伤在规定的医疗期内
- 女职工在孕期、产期、哺乳期内
- 在本单位连续工作满15年且距法定退休年龄不足5年
- 法律法规规定的其他情形

但这些人如果严重违纪（第39条），公司仍然可以解除，且不需要给补偿金。

## 七、证据清单

仲裁最重要的就是证据，提醒用户尽早收集：

**必须收集**：
- 劳动合同
- 工资条 / 银行流水（近12个月）
- 解除劳动合同通知书（书面）
- 考勤记录

**有就更好**：
- 工作群聊天记录（微信、钉钉）
- 工作邮件
- 录音（与HR、领导谈话）
- 社保缴费记录
- 加班记录
- 年假未休记录
- 绩效考核结果

**证据保全**：
- 截图保存，不要只留在公司电脑
- 聊天记录不要删
- 录音注意：自己参与的对话可以录，不需要对方同意
- 工作文件用个人邮箱发一份给自己

## 八、免费法律援助

- **12333** — 人力资源社会保障热线，可咨询劳动法问题
- **12348** — 法律援助热线，符合条件的可免费指派律师
- **当地总工会** — 有免费劳动法咨询
- **司法局法律援助中心** — 经济困难可申请免费律师
- **高校法律援助中心** — 部分法学院有免费劳动法咨询

## 九、重要时效

- **劳动仲裁时效**：1年（从知道或应当知道权利被侵害之日起）
- **拖欠工资**：劳动关系存续期间不受1年限制，离职后1年内申请
- **对仲裁裁决不服起诉**：15日内
- **对法院一审判决不服上诉**：15日内
