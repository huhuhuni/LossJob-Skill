# 失业 Skill

帮助失业人群的 AI Skill，提供**裁员维权**和**心理支持**两个核心功能。兼容 Claude Code、OpenClaw 等本地 Agent。

## 功能

### 裁员维权

- **赔偿金计算** — N、2N、N+1，根据你的情况算出具体数字
- **利弊权衡** — 帮你分析要不要仲裁，而不是无脑怂恿
- **仲裁流程指导** — 从收集证据到开庭到执行，每一步怎么做
- **劳动法解析** — 哪些辞退是违法的，哪些钱你该拿没拿
- **赔偿金税率** — 绝大多数人的赔偿金在免税范围内
- **免费法律援助** — 12333、12348、总工会、司法局

### 经历回应

根据你的处境分层给支持——崩溃区只保安全，焦虑区只给最小动作，行动区才谈策略。

| 类型 | 你的状态 | 核心策略 |
|------|----------|----------|
| A | 刚失业，有点慌 | 情绪缓冲 + 最小保底动作 |
| B | 投了几个月没回音，开始自我否定 | 切断错误归因 + 强制换动作 |
| C | 存款快没了，撑不住了 | 100% 生存现金流 |
| D | 行业没了，不知道还能干吗 | 能力平移 + 低学习成本路径 |
| E | 觉得活着没意思 | 停止一切求职建议 + 专业心理资源 |

E 类心理危机优先级最高，一旦识别到相关信号，立即停止求职建议，只给心理安全支持。

## 安装

把仓库地址给 Agent，让它自己装：

```
https://github.com/huhuhuni/LossJob-Skill
```

支持 Claude Code、OpenClaw 等支持 Skill 安装的 Agent。

## 使用

安装后，以下说法会自动触发 Skill：

- "被裁员了，赔偿金只给了N"
- "投了三个月简历没回音"
- "存款快没了不知道怎么办"
- "公司调岗降薪逼我走"
- "感觉活着没意思"

也可以直接输入命令：

```
/unemployment:unemployment 刚被裁了不知道怎么办
```

### 不支持 Skill 安装的 Agent

对于 ChatGPT、豆包、Kimi 等不支持 Skill 的平台，将 `system-prompt.md` 全部内容复制到自定义指令（Custom Instructions）即可。`system-prompt.md` 已包含所有法律依据、仲裁流程和回应策略，无需额外加载文件。

## 示例对话

**维权场景**：

> 用户：公司今天通知我被裁了，只给N，干了3年，月薪1万2，这合理吗？

> Skill：干了3年，月薪1万2，N的话是3.6万。但要看公司是怎么裁的——如果是经济性裁员或者协商解除，N是合法的底线。如果是没有任何合法理由直接辞退，你可以主张2N，也就是7.2万。公司有没有给你书面通知？上面写的离职原因是什么？

**支持场景**：

> 用户：投了三个月简历没回音，感觉像个废物

> Skill：你已经在很努力地找了，这个努力本身就值得被看见。投简历没有回音，不是你的问题，是渠道的问题。大多数招聘系统本身就是黑洞。我们先停掉"天天投简历"这件事。今天停投简历，只做一件让自己有掌控感的小事——出门走20分钟、做一顿饭、或者联系一个不求职的朋友。

## 项目结构

```
LossJob-Skill/
├── .claude-plugin/plugin.json          ← Claude Code 插件声明
├── skills/unemployment/
│   ├── SKILL.md                        ← Claude Code Skill 入口
│   ├── system-prompt.md                ← 通用系统提示词（自包含，任何 Agent 可用）
│   ├── README.md
│   └── references/
│       ├── labor-law.md                ← 劳动法、赔偿金、税率、证据
│       ├── arbitration.md              ← 仲裁流程、开庭、执行
│       └── strategies.md               ← 经历回应详细策略
└── .gitignore
```

> `system-prompt.md` 已内联 references 的全部内容，可直接粘贴使用。`references/` 目录供 Claude Code 的 SKILL.md 动态引用，或作为独立参考资料查阅。
